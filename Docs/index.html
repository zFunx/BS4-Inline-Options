BS4 Inline Options v0.0.0

Extends Bootstrap 4 CSS for creating horizontal radios
Setup
Link the following styles:
<!-- Bootstarp CSS -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.0/css/bootstrap.min.css">

<!-- Inline options CSS -->
<link rel="stylesheet" href="inline-options.css">

Inline Options
Add .inline-options class with the parent .input-group element
<div class="input-group mb-4 inline-options">
    ...
</div>
With prepended text 
<div class="input-group mb-4 inline-options">
    <div class="input-group-prepend">
        <span class="input-group-text">How much stars would you like to give to this channel?</span>
    </div>
    ...
</div>
How much stars would you like to give to this channel?
.input-group is actually a block element. To make it inline, add classes .d-inline-flex and .w-auto 
<div class="input-group mb-4 shadow-sm d-inline-flex w-auto inline-options">
    <div class="input-group-prepend">
        <span class="input-group-text">How much stars would you like to give to this channel?</span>
    </div>
    ...
</div>
How much stars would you like to give to this channel?
